# portesouvertes

Sur Linux
```python
python3 -m venv env
source env/bin/activate
python3 -m pip install -r requirements.txt
python3 main.py
```

Sur Windows
```python
python -m venv env
Set-ExecutionPolicy -ExecutionPolicy Undefined -Scope CurrentUser
.\env\Scripts\activate
python -m pip install -r requirements.txt
python main.py
```
